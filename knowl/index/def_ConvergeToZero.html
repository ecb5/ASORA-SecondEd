Definition 6.1.1.

Let \(\left(s_n\right)=\left(s_1,s_2,s_3,\ldots\right)\) be a sequence of real numbers. We say that \(\left(\boldsymbol{s}_{\boldsymbol{n}}\right)\) converges to 0 and write
\begin{equation*} \limit{n}{\infty}{s_n}=0 \end{equation*}
provided for any \(\eps>0\text{,}\) there is a real number \(N\) such that if \(n>N\text{,}\) then \(|s_n|\lt \eps\text{.}\)
