Definition 10.4.2. The Riemann Integral.

Given a function \(f(x)\) defined on the interval \([a,b]\text{,}\) we say \(f\) is integrable on \([a,b]\) if and only if there is a number \(I\) such that for each \(\epsilon >0\text{,}\) there is a \(\delta >0\) such that for any partition \(P=\{x_0, x_1, \cdots, x_n\}\) of \([a,b]\) with \(\norm{P}\lt\delta \text{,}\) we have
\begin{equation*} \left|\sum^{n-1}_{k=0}{f\left(x^*_k\right)\left(x_{k+1}-x_k\right)}-I\right|\lt\epsilon \end{equation*}
for any choice of \(x^*_k\) where \(x_k\le x^*_k\le x_{k+1}\text{.}\)
