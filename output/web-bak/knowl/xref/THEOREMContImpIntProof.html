Sketch of Alleged Proof.

Note that we’re calling this an “alleged” proof. That means that it contains a flaw somewhere. As you read it see if you can find where it goes wrong.
We already know that
\begin{equation*} \underline{\int^b_{x=a}}{f(x)}\dx{x}\le \overline{\int^{b}_{x=a}}{f(x)\dx{x}} \end{equation*}
If we can also show that
\begin{equation*} \overline{\int^{b}_{x=a}}{f(x)\dx{x}}\le \underline{\int^b_{x=a}}{f(x)}\dx{x} \end{equation*}
then the conclusion follows immediately.
Let \(\eps \gt 0\) be given. Since \(f\left(x\right)\) is continuous at each point in \([a,b]\text{,}\) we can choose a \(\delta >0\) such that if
\begin{equation} \left|x-y\right|\lt \delta \text{,}\tag{12} \end{equation}
then
\begin{equation} \left|f\left(x\right)-f\left(y\right)\right|\lt \frac{\eps }{b-a} \text{.}\tag{13} \end{equation}
Since \(f\) is continuous on \([a,b]\) it is continuous on each subinterval. Next define \(m_k\) and \(M_k\) to be the respective minimum and maximum of \(f(x)\) on the subinterval \([x_k, x_{k+1}]\text{.}\) If we choose a partition
\begin{equation*} P_0=\{x_0, x_1, x_2,\dots , x_{n-1}, x_n\} \end{equation*}
such that \(\norm{P_0}\lt \delta \text{,}\) then on each subinterval \([x_k, x_{k+1}]\text{,}\) we have
\begin{equation*} M_k-m_k\lt \frac{\eps }{b-a}\text{.} \end{equation*}
Thus
\begin{align*} \overline{\int^{b}_{x=a}}{f(x)\dx{x}}-\underline{\int^b_{x=a}}{f(x)}\dx{x}\amp=\inf_{P} \left(U(P)\right) -\sup_{P} \left(L(P)\right)\\ \amp{} \le U\left(P_0\right)-L(P_0)\\ \amp{} =\sum^{n-1}_{k=0}{M_k\left(x_{k+1}-x_k\right)}-\sum^{n-1}_{k=0}{m_k\left(x_{k+1}-x_k\right)}\\ \amp{} =\sum^{n-1}_{k=0}{\left(M_k-m_k\right)\left(x_{k+1}-x_k\right)}\\ \amp{} \lt \sum^{n-1}_{k=0}{\frac{\eps }{b-a}\left(x_{k+1}-x_k\right)}\\ \amp{} =\frac{\eps }{b-a}\sum^{n-1}_{k=0}{\left(x_{k+1}-x_k\right)}\\ \amp{}=\frac{\eps }{b-a}\left(b-a\right)\\ \amp{}=\eps \end{align*}
QED?
