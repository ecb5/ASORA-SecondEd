Definition 10.2.2. The Derivative, An Alternative Definition.

Given a function \(f(x)\) defined on an interval \((a,b)\text{,}\) and a point \(x\in (a,b)\text{,}\) the derivative of \(f\) is given by
\begin{equation*} f^\prime(x)=\limit{y}{x}{\frac{f(y)-f(x)}{y-x}}\text{.} \end{equation*}
